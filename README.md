# Central Vintage

Central Vintage is an unofficial addon for Create: Central Kitchen and Vintage Delight.
It adds a Pickling Guide for Vintage Delight fermenting recipes, allowing Create: Central Kitchen blaze-stove guide automation patterns to work with Vintage Delight fermenting jars.
The Pickling Guide also links to Create: Central Kitchen's existing blaze-stove Ponder scenes for in-game guidance.

## Requirements

- Minecraft 1.20.1
- Forge 47.4.10 or newer
- Create
- Farmer's Delight
- Create: Central Kitchen
- Vintage Delight

## Status

This project is in development.
The current item and GUI textures for the Pickling Guide are blank placeholders. Final assets should be original work made for this project, while referencing the general Cooking Guide layout and automation concept from Create: Central Kitchen.
